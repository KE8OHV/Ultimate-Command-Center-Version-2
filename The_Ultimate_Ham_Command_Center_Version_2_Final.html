Quick Start + How To Use Everything

Built-in instructions were added so other hams can learn the program quickly without guessing what each area does.
10 quick guides

1. SDR + Frequency Control

Choose an SDR provider, type a frequency or use the keypad, select the mode, then press Jump Now. Use Open SDR if a public receiver refuses to show inside the frame. WebSDR Twente is now restored as a direct option like the older command center.

2. CAT /
Rig Control

Pick a rig profile and baud rate, connect with CAT / Serial, then use Send to Rig from the SDR section. The browser must support Web Serial and the radio must be set up for CAT. If direct CAT is not available, the program still works as a control dashboard.

3. HF Nets

Click a listed net to load its frequency and mode. Then press Send to Rig or Open SDR. Use Add Custom Net to build your own local net list and save notes for check-ins, hosts, or schedules.

4. Propagation Planner

Enter Solar Flux, A index, K index, and time of day, then run the planner. This gives a practical suggestion for stronger bands and operating windows. The external solar tools below open deeper live resources.

5. Logging

Add call, band or frequency, and notes into the log table. Export ADIF when you want to move contacts into another logging program. Import ADIF if you want to reuse saved data in this browser.

6. Equipment Inventory

Use the inventory database to save radios, amps, tuners, microphones, antennas, power supplies, computers, and accessories. Export CSV when you want a simple backup or printable list.

7. Node / ClearNode Listening

Paste a stream or listener page URL, then load it. If the feed is a direct audio stream, the player may work right inside the page. If the feed is a web page, use Open Listener in a new tab.

8. Antenna + SWR Tools

Use the built-in wire calculator, wavelength helper, and SWR estimator for quick checks. Save antenna notes for feedpoint tests, wire lengths, tuner settings, and matching results.

9. Ham Tools Area

The tool launcher opens propagation, DX, SDR, logging, learning, map, timing, AllStar, EchoLink, emergency, and station references. Every card includes at least one working action button or an internal jump link.

10. Accessibility + Profiles

Toggle Accessibility Mode for easier navigation, stronger contrast, and optional speech. Use Export Profile Backup to save memories, notes, and inventory from this browser. Import Profile restores them later.

    Built-In Ham Radio 101 Guide

    This is an original built-in reading guide based on the same beginner topics people usually look for on ARRL's Ham Radio 101 pages, but kept inside your Command Center so it always opens correctly from the program.
    TopicQuick Guide
    What is ham radio?Ham radio is licensed two-way radio communication on amateur bands using voice, Morse code, digital modes, images, satellites, and more.
    Why people enjoy itOperators enjoy learning radio, talking locally and worldwide, emergency communications, portable operation, contests, satellites, and building stations.
    Who can be a ham?People of all ages and backgrounds can participate. The hobby includes beginners, experimenters, public-service volunteers, DXers, ragchewers, and digital-mode operators.
    How to get startedStudy for the FCC exam, earn a license, choose a simple first station, and get on the air with help from local clubs, mentors, and online resources.
    Why a license is neededAmateur radio uses shared spectrum with operating rules, so operators are tested on basic regulations, safety, and operating knowledge before transmitting.
    What it costsYou can start small. Many operators begin with study material, a basic handheld or starter station, and add gear over time as their interests grow.
    Morse codeMorse code is still popular, but it is not required for a U.S. amateur radio license.
    Amateur bandsHam bands cover many frequency ranges. Different bands support local, regional, worldwide, satellite, and experimental communication.
    Other communication modesBesides voice, hams use CW, RTTY, PSK, FT8, FT4, JS8Call, packet, image modes, APRS, satellites, and other digital techniques.
    Getting involvedJoin a club, check into nets, operate special events, try POTA or SOTA, volunteer for emergency communications, and keep learning.
    Best next stepUse the built-in glossary below, the phonetic trainer, and your command center tools to learn terms, practice callsigns, and explore operating.
